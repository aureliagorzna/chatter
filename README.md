This is a generic chatting app using bcrypt to hash passwords and JWT to authenticate users.

- beta version

HOW TO RUN

Download the repo and run those commands:

- cd backend && npm i
- npm run start
- continue in another console
- cd ../frontend && npm i
- npm start
- go to http://localhost:3000
